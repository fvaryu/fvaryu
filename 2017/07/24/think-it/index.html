我们为什么会反感培训出来的程序员？

发表于 2017-07-24 | 分类于 随笔 | | 阅读次数

2017/7/25补充

此文并非是来拉仇恨，鄙视任何人的，而是分析这种现象的机理的。我只是一直不理解为什么会有这种现象，下文是我的一些思考。

我们为什么会反感培训出来的程序员？

在IT开发这个行业至少有一半以上的人是通过培训走上这个行业，这也就导致了一些企业或者开发着鄙
视看不起这一类人群，我一直在想培训出来的就没有牛人嘛，答案是肯定，就像上学那会成绩好的也
不只是存在于重点班级，总会有一部分异类(非贬义，特指在特定人群中比较突出的部分)，IT培训业同样有优秀人才，那么我们到底在鄙视、抵制什么呢？

诱因

前两天一个高中同学（高中毕业10年，没有上大学，高中毕业出去学习机械加工，一直在工厂干活）突然联系我，问我Visual C++6.0怎么安装，他一直管不好，一下把我问懵逼了，我在想机械加工厂要用这个干嘛。了解后才知道这些年工厂干活不易，生活压力（不久前有了小孩，月工资5000左右）大了，觉得不挣钱，没有什么前途，听说IT工资（他一个亲戚，大学毕业，IT行业，月薪20000的样子，真伪不知）不错就像，转行写代码，所以才有了上面的问题。

瞎扯一会。

下面是他问我的一些主要问题：

  • 关键问题一：

    你是码农还是程序猿？

  • 关键问题二：

    是否包吃住？收入怎么样？

  • 关键问题三：

    多久加薪？晋升空间如何？

  • 关键问题四：

    是自学还是去培训？

我不知道你从这个几个问题看到或想到了什么问题。我想这种现象在想要入行IT的人群中肯定不在少数。同样的你当初入行的时候是不是这样的呢？
从何他对话中反应的问题：

  • 问题一：

    他不了解这个行业，对自己的定位不清晰。

  • 问题二：

    只是因为收入高，所以就想进来分一杯羹。

最后我建议他（以这么多年对他的了解），不要心急，给自己1-3个月的时间充分了解IT这个行业之后再做打算，同时看一写基础的东西，比如大学都会学的计算机基础、C语言等，另外就是分析一下除了这些年制造业不景气以外，自己为什么没有大的进步，10年时间生活依旧如此的原因，最好是一条一条罗列出来，无论它的影响大小。

我不能保证自己的建议是否100%正确，至少能够让他做到不盲从，在我看来这是最优的方案了，然而他并没有采纳我的建议，而是选择辞职参加培训。

到这里我不知道各位读者有什么看法和想法？仔细分析一下你会看到，他来咨询只是寻求一种心理安慰，让他更有底气来做，即使不成功后面也有更好的借口来推卸失败的责任，所以说既然已经选择何必来问呢。

说点什么

前段时间看到一个报道互联网这块有300、400万的岗位需求，我不知道数据的真实性有多少，但是我咨询了一些从事招聘工作的朋友，据他们反馈，大部分公司都缺人，简历源源不断，但是匹配度极低。偶尔一个符合度高的简历都抢着要（HR会互相推荐应聘者）。

我想读者对于招聘难，应聘那这种现象也有耳闻，是什么原因导致的呢？

鄙视的不是培训以及培训出来的人

培训对于任何一个人或企业都是必不可少的。你是否在刚刚走出大学校园的时候，把企业有定期培训的作为标准之一呢？培训实际上是我们提升自己职业技能的有效途径之一。同样的我们想要进入一个未知领域，首选的就是培训。这叫好比想要学习知识要找老师，考驾照要找教练一样。那我们鄙视什么？

鄙视的是培训机构以及自我

培训机构

我差不多有1个月的培训经历，当时在武汉，算是半个培训出来的人。在这个把月的时间里负责人的说我确实学到了东西，但是这个东西就像是码砖一样，把成型砖（知识点）塞给我，然后我一块一块整齐的堆砌在我的脑子里。这种形式的教育我们经历的太多，同样我认为这种对于固定式的编码结构来说也是最有效的。既然如此培训机构的问题出在哪里呢？我认为它的恶至少有两点（或更多）：

  • 其一
    不考虑学员本身的素质(不是那个素质，是否有编程能力)，只要给钱就可入学
  • 其二
    虚构能力，虚构项目，无节操的装饰简历
    自我
    说明：此观点，纯属作者自己瞎扯淡。
    为什么这里要说到自我，即已经在行业里面的人。因为在我看来出来外在的因素，还有部分是来自于我们自身，你可以反驳我但是，请你仔细想一想在怼我。
  1. 自己不自信，能力欠缺，引发的恐惧
    鄙视或抵制培训出来的人，本质上也体现了一种自我保护，保护什么？保护我们自己的饭碗。因为这么多人都通过培训找到了工作，而且工资都还不错，那么给我们带来的是什么，对，就是竞争，人越多竞争就越大，机会就越少，在加上自身的能力问题，那么很有可能自己就丢到了饭碗。所以说在我看来，看不起培训出来的人实质上也是看不起自己，不敢相信能够在激烈的竞争中出类拔萃。
  2. 莫名其妙的优越感（区分我和他）
    我想这一条很多人都有感受，当我们说我是科班出身，他是培训出身，内心是什么状态，一种优越感油然而生对不对？而这种现象实际上是在区分我和他，划一道界限，本质上想表达什么？我比你厉害，你只配给我提鞋，提鞋的你就不要进来了。

我们需要的是能力不是搬运工

对于上面我讲的那个同学，我不知道他未来的路什么样的，但是我知道程序员的路应该是什么样的，我们需要的是能力不是搬运工。
上面的一个分析和观点，只是就着单纯的现象而言，或许讲述不够准确，但是也剖析了一些本质。
不管是科班、自学还是培训出来的程序员，这都是走上这条路的手段和途径。我们无需过多关注这些外在的东西，种在花盆里的花同样花开花不是吗？我们需要重视的是如何让花盆里的花枝繁叶茂而已。
