当我年少

发表于 2016-07-14 | 分类于 随笔

当我年少，我就在想我会成为怎样的人，会有怎样的生活，而今年逾而立，我依旧在想我会成为什么样的人，以后会有怎样的生活。人生就是这样你永远不可能知道以后会怎样，但你总是在想，在寻找，在向前，即使你意欲停留也会有一个叫做时间的影子推着你向前，一刻也不停留。

不知你是否发现，你的人生轨迹总是偏离预想轨道，人生规划也朝着相反方向发展。昨日你努力读书，希望有一个优异的成绩，考上梦寐以求的学校，邂逅一纸浪漫爱情，获取一份理想职业，成就一个完美人生。然而路总是越走越弯，辛勤努力换来的不过是三流学校，孤单身影，奔波劳碌，以及一个暗无天日的人生罢了。

——–hello-pic.jpg——–

当我年少，我放飞了写着梦想的纸飞机，那时以为天空广阔总是要去追寻。飘落，便是风景。而今再也没有那种心境与闲情，有的只是对未来的彷徨和恐惧。看着楼下穿梭的车辆与匆匆的行人，你是否有些伤感，那个爱幻想的年纪天真的面孔渐悉模糊消逝，你却无能为力，留下的是为生计匆匆而过的背影。

也许有天你该抽个时间出去走走，去山涧，去村庄，去农田，吹吹那里的风，看看那里的绿，坐在田埂上数一数山坡上的羊，捉一只蛐蛐听一听那清脆的叫声。也许你还会遇见曾今的你，不知那时你是否会流泪。

当我年少，我期望每年的生日，那天或许是当时最美好的一天。因为那天可以得到最想要的东西。是的，那时有太多的想要，也只能寄希望于一天从众多想要中挑选最期待的了。所以总是盼望着快快长大，成为那个可以随心所欲的人。而全然不顾明天会怎样。这也许是唯一实现了的梦了吧。而如今我想要的仍然太多，却再也没曾经的美好。By lowett

Android`s，面试不过也许真的是你不够努力

发表于 2016-07-13 | 分类于 面试

最近公司新项目启动需要招人，有幸本人负责安卓技术这一块，因为本人也是菜鸟，哈哈

陆陆续续公司来了一些面试者，但是到目前为止还没有一个合适，不是说他们的技术不过关，而是里面参杂着形形色色的问题。下面说说自己的一点想法。
如果说的不对的地方还请谅解，谢谢。

一、做一点准备工作

作为一个应聘的去参加面试前势必要对公司有些了解，既然决定去面试就说明自己认可或者能够满足自己的需求现状，那去之前是否应该多少了解一点公司的情况，不然不就是浪费大家的时间了吗。我面试的人中大部分对公司是做什么的完全不知道，简历也是海投的，更有甚者公司名字都不知道，那你还面试个啥呢。

二、合理的薪资要求

高薪大家都喜欢，但是是不是应该务实一点呢。收到的简历中薪资要求都是异常的高，只有一个姑娘－－硕士研究生，薪资要求比较合理的，根据自己的能力合理的定位，之所以最终没有要她，是我觉得她应该去大一点公司做研发比较合适，我们只是一个简单的App，她的能力以及职业发展方向不能够和公司对应起来，对她对公司都是一种损失，而其他的面试者基本是瞎要，不能给自己一个合理的评估。刚出道做了，哦，不对，是参与了一个项目之后就多少K多少K，而技术呢，一塌糊涂。有人会说了时我要求太高了，在这里要说明一下，我问的全都是基础，因为我们也不是招很厉害的高级工程师，能够满足基本的业务需求就行。

三、提升自己的技术能力

面试的过程中发现，多数应聘者都是停留在会使用api的层面，稍稍深的一些实现都没有去看看，对，是看看，研究那就更是寥寥无几了。比如有一个应聘者说Okhttp精通，然后我就问Okhttp支持SDPY吗，问：SDPY是啥，没听过，令我汗颜，SDPY可是OKHttp的一大特点，这都不知道还精通个啥，更别说底层一点的实现了。对于大多数的app开发者来说，所有的东西都精通确实是没有必要，但是自己用的技术是否应该了解清楚一点呢，不然怎么能在开发中轻车熟路呢，怎么吸取经验提升技术呢？

四、不卑不亢展示自信的自己

我不知道是我长的太凶还是怎么的，有一位应聘者，面试中声音都在颤抖，紧张的不行，而且还是有一定开发经验的。面试中我基本是保持一种交流的心态，而不是一个考官的姿态，基本上是根据应聘者提供的项目经历问问团队情况，充当的角色担任的职责以及项目中使用到的技术。所以平日开发中还是要对用到的做一个深层次的了解才行，只有这样才能充满自信展示自己，面试官不可能通过短短的几十分钟，清楚的了解你的所有情况，只有展示真实的自我，把用到说出来就行了。有些偏的东西即使回答不出了，那在面试官看来也是可以理解的，说白了没有哪一个面试官能够掌握所有技术。

五、努力吧，少年

安卓开发的局限还是比较大的，发展空间也比较窄，所以呢要么转行，要么做深，而现在混日子的一抓一大把，能力的提升不会随着工作年限的增长而增加，当你的能力跟不上工作年限的时候势必会被淘汰，当你拿着有4、5年工作经验的简历，在面试中连一个基本技术点答不上来的时候是否会惭愧，是否会脸红？

这些对一个普通的开发者都是日常，会用到的东西，或者是间接会接触到，如果连这都不能很好的说个一二三，还有什么可谈的呢？Android`s加油吧！

聪明的人都在思考奔跑，平庸的人还在呻吟

发表于 2016-07-12 | 分类于 随笔

“聪明的人都在思考奔跑，平庸的人还在呻吟”，当你读到这句话的时候不知你作何感想，也许你会骂我矫情，也许你会停下思考，但是不管怎样这句话都代表了一种社会现象，也许只是我眼中的。

——–明天你好-pic.jpg——–

前段时间和朋友一起闲坐交谈，他抱怨毕业这些年分文未存，而房价又蹭蹭的一直往上涨，工作也是朝九晚五，很是辛苦，迷茫不知所措，看不到前方的路，未来在何方，等等。听他发完牢骚，我沉默不语，一时间竟然找不到用什么话语来安慰他。只好什么也不说，我俩就这样静静地坐了好久，各自思考着。

对于我来说，他时值得我去羡慕的，毕竟是从小有名气的院校毕业又有学位，还有一份在我们外人看来不错的职业。而我呢，只是一个大专毕业自学本科罢了，有时候我都不愿和别人谈论学校，毕竟是一所不知名学校，说出来大家或许还会惊奇居然有这么所院校的存在。的确，这就是现实，出自泥里的人总会是带着一股泥腥味伴着你，使你喘不过气。

曾经也奢望过要去一所高校深造，武装丰满自己以便能更好的适应这个极速发展的社会，然而事与愿违，现实总是残酷的，梦破碎的那刻总有几珠泪光闪过。也许你会觉得故事到这里就完了，他的人生注定会光彩夺目，平步一生，而我就是沦落到市井小镇平凡而过。那你错了，就像那句话所说的，上帝为你关闭一扇门就会给你打开一纸窗，为你打开一扇门自然会为你关闭一纸窗。

——–天空遥望-pic.jpg——–

事实上是，这些年下来我看到的并不是他的一路好走，而是皱起的眉头，暗淡的眼神，还有那个压弯的肩膀。因为他的起点自然想到的就是应该有的终点，而不曾停下来去思考努力的方向。这样注定会沦落到平庸的阶层，去抱怨这个膨胀的社会，去控诉它的不公。但你是否想过成功该有的付出和心酸。哪怕是再优秀的底子也要继续向前才行呀。

当你羡慕谁谁谁，怎样怎样，自己如何不好的时候是否想过，在这个竞争激烈的社会，谁又容易呢？公务员？公司高管？下岗工人？是的，没有一个可以顺风顺水，有的只是对明天的期待与生存下去的欲望，加上那个从不停止跳跃，想出来的心。

曾经有一段时间，我也很是迷茫，不知所措，看着别人风生水起，自己只能暗自着急，一遍又一遍的告诫自己要努力，要加油，努力就有出路。努力谁都会说，谁都会写，但是谁又能讲清楚其中的意义呢？努力的意义不仅仅是停留在一句口号之上，更多的是要去思考，去实践，当你试着去思考彼此之间的差异，总结出规律现象的时候你就有了努力的方向，也就知道了需要提高的问题所在。当你有了这样的观念的时候，暗淡下去的心，自然会充满激情与自信。

在各种功夫电影里，高手从来都是以不变制万变，他胜利的诀窍不是大声吆喝来斥退对方，而是观察对手的一举一动，思考反击对策，一击致命。在这个浮躁的年代，有太多的呻吟与抱怨，是否应该停下来，思考一下，买衣服的时候还要货比三家，考虑一下哪家更实惠，而面对更长的人生道路却丧失这种天然本能，是不是一种悲哀呢。

奋斗未来，我正年轻。让思考成为一种习惯，艰难困苦没有人能够帮助你，优秀也从来不是与生具来的，只有自己能够成就自己。

开篇

发表于 2016-07-10 | 分类于 随笔 
生活不止眼前的苟且，还有诗和远方的田野，
你赤手空拳来到人世间,为找到那片海不顾一切
                        --《生活不止眼前的苟且》 许巍

当你抱怨生活，盛赞他人如何如何的时候是否有思考别人是怎样一步步走来的，优秀从来都不是与生俱来的，当你睡觉玩乐的时候别人还在努力！正所谓自助者天助，艰难困苦也唯有自己能帮你了！既然不能承担应有的付出，那便无法享有美好的以后！

互联网时代，优秀的团队太多，优秀的人才也多，一直想成为行业的先锋，但是从来都是停留在想想上，没有行动起来。今天决定真正的开始做起来，把自己的经历记录下来，不仅仅是对自己生活的感悟，同时也许能够给迷茫的你一点启示。

文笔虽不好，但是：说我所想，写我所看。
